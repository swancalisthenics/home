Datenschutzerklärung

1. Verantwortliche Stelle

Verantwortlich für die Datenverarbeitung auf dieser Website ist der oben im Impressum genannte Community.

2. Hosting über GitHub Pages (Datenübertragung)

Unsere Website wird als statische Seite über den Dienst „GitHub Pages“ der Firma GitHub Inc. (88 Colin P. Kelly Jr. St, San Francisco, CA 94107, USA) gehostet und bereitgestellt.

Wenn Sie unsere Website aufrufen, stellt Ihr Browser eine Verbindung zu den Servern von GitHub her. GitHub erhebt dabei automatisch Protokolldaten (sogenannte Server-Logfiles). Dazu gehören:

  • IP-Adresse des zugreifenden Geräts
  • Datum und Uhrzeit des Abrufs
  • Übertragene Datenmenge und Name der abgerufenen Datei
  • Browsertyp und Version sowie das Betriebssystem Ihres Geräts
  • Referrer-URL (die zuvor besuchte Seite)

Zweck dieser Verarbeitung ist die sichere, stabile und fehlerfreie Bereitstellung unserer Vereinswebsite. GitHub speichert diese Daten aus Sicherheitsgründen. Die Datenverarbeitung steht im Einklang mit dem Schweizer Datenschutzgesetz (DSG). Weitere Informationen finden Sie in der Allgemeinen Datenschutzerklärung von GitHub sowie in den spezifischen Hinweisen zur Datenerhebung bei GitHub Pages .

3. Cookies, Tracking und Registrierung

Diese Website dient ausschliesslich der Information über unsere Community. Wir betreiben kein Backend, nutzen keine Cookies und verwenden keine Webanalyse- oder Tracking-Tools (wie Google Analytics). Es gibt keine Registrierungs-, Login- oder Kontaktformulare. Es werden von uns keine personenbezogenen Daten aktiv abgefragt oder gespeichert.

Auf unserer Website veröffentlichen wir Bildmaterial von Communityaktivitäten und den Communitygründern. Dies geschieht auf Grundlage des berechtigten Interesses der Community an der Öffentlichkeitsarbeit sowie mit dem Einverständnis der abgebildeten Personen. Sollten Sie mit der Veröffentlichung eines Bildes, auf dem Sie erkennbar sind, nicht einverstanden sein, genügt eine kurze Mitteilung an die oben genannte E-Mail-Adresse, und wir werden das Bild umgehend entfernen.

Die Verweise auf soziale Netzwerke (z. B. LinkedIn, Instagram) sind als rein statische Verlinkungen ausgestaltet. Das bedeutet: Beim reinen Besuch unserer Website werden keinerlei Daten an diese externen Plattformen übertragen. Erst wenn Sie aktiv auf einen solchen Link klicken, verlassen Sie unsere Website und werden auf die Plattform des jeweiligen Anbieters weitergeleitet. Ab diesem Zeitpunkt gelten die Datenschutzbestimmungen der jeweiligen Plattform.
